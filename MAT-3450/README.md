# Projet final de modélisation mathématique

Ce projet a été réalisé dans le cadre du cours d’introduction à la modélisation mathématique (MAT-3450). Suite à la polémique entourant l’abattage de 15 cerfs dans le Parc Michel-Chartrand, notre objectif était d’évaluer l’effet de la variation du nombre de cerfs sur le risque de contracter la maladie de Lyme. Pour ce faire, nous avons modéliser la croissance d’une population de tiques à leurs différents stades de développement et selon leur statut d’infection à la Borrelia burgdorferi, bactérie responsable de la maladie de Lyme, en fonction du temps et du facteur biotique de parasitisme chez le cerf.

Auteurs: Simon-Olivier Laperrière, Shophika Vaithyanathasarma et Noémie Chenail

## Répertoire

Le fichier `projet.ipynb` contient le code utilisé pour modéliser la dynamique de la population de tiques, ainsi que pour générer les graphiques du rapport. Ce code est également disponible sur [Google Colab](https://colab.research.google.com/drive/1f1d__tdHM6BJtdDLhjey79bfe5Ce459y?usp=sharing).
